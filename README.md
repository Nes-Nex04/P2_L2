# P2_L2 - Práctica de CSS (Box Model / box-sizing)

Este proyecto es una práctica donde aplicamos conceptos básicos de CSS como **box-sizing**, **box model**, **margin**, **padding**, **border**, **color**, **font-family** y **text-align**.

## 🧪 ¿Qué contiene?

- `index.html`: una página de ejemplo ambientada en F1 con varias tarjetas y bloques.
- `styles.css`: estilos que muestran cómo funciona `box-sizing: border-box` y cómo se compara con `content-box`.

## ▶️ Cómo ejecutar

Abre `index.html` en tu navegador preferido (doble clic o arrástralo al navegador).

## 📝 Qué puedes modificar

- En `styles.css` prueba a cambiar los valores de `padding`, `border` o `width` de las tarjetas y observa cómo se comportan cuando usas `border-box` vs `content-box`.
- Edita el contenido de `index.html` para agregar más secciones o ejemplos.
